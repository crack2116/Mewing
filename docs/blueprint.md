# **App Name**: ShipShape Dashboard

## Core Features:

- Dashboard Overview: Display key metrics like total revenue, services in progress, completed services, and pending requests in an easy to view format.
- Recent Service Requests: Show a list of recent service requests with details like client, status, and date.
- Quick Actions: Provide quick access to common tasks like creating a new request, managing clients, and real-time tracking.
- Intelligent Status Prediction: Employ an AI tool to predict the status of pending service requests based on historical data and current conditions.  This may suggest proactive interventions.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) for a professional and trustworthy feel.
- Background color: Dark navy (#0F172A), desaturated for a modern dark theme.
- Accent color: Purple (#7C3AED), contrasting for highlighting key elements and interactive components.
- Headline font: 'Space Grotesk' sans-serif for headlines and short amounts of body text; for longer text use 'Inter'.
- Body font: 'Inter' sans-serif for body text to ensure readability and a clean interface.
- Use consistent, clean line icons to represent different sections and actions.
- Dashboard layout with clear sections for key metrics, recent requests, and quick actions.
- Subtle animations and transitions to improve user experience.
